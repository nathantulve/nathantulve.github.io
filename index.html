CSS Waffle

CSS Waffle is a fully code generated typeface. The initial idea was inspired by Yusuke Sugomori's CSS Sans. The typeface is fully customizable.

    How to use:

  • Download CSS
  • Link it up with Html

    • <link rel="stylesheet" type="text/css" href ="css_triangle.css" />

  • Write the letters in Html

    • <div class="letter A">▶︎</div>

    <div class="letter B">▶︎</div>

    <div class="letter C">▶︎</div>

    ...

  • Enjoy!
